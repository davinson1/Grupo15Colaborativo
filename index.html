Objetivos del curos

Generrales:
Operar lenguajes de Diseño web como HTML5 y CSS3.

Objetivo específico 1:
Desarrollar Sitios Web que satisfagan las necesidades del usuario.
Objetivo específico 2:
Introducción a HTML5, Estructura básica de HTML5, Aplicación de elementos de HTML5 a un proyecto de diseño Web.
Objetivo específico 3:
Introducción a CSS3, Estructura básica de CSS3, Aplicación de elementos de CSS3 a un proyecto de diseño Web.


Objetivos del tema seleccionado:

Generrales:
Información complementaria sobre GIT/GITHUB, HTML5 y CSS3.

Objetivo específico 1:
Introducción, profundización e información complementaria sobre el uso de GIT y GITHUB.
Objetivo específico 2:
Gestionar proyectos utilizando GIT y GITHUB.
Objetivo específico 3:
Gestionar proyectos colaborativos de diseño web, HTML5 y CSS3.

Temática

Los sistemas de control de versiones son programas que tienen como objetivo controlar los cambios en el desarrollo de cualquier tipo de software, permitiendo conocer el estado actual de un proyecto, los cambios que se le han realizado a cualquiera de sus piezas, las personas que intervinieron en ellos, etc. Entregar información al usuario sobre el desarrollo de sitios web utilizando GITHUB/HTML5 y CSS3.
